DCPU-16 emulator
================
**Alpha stage**
----------------

Already works:
 - emulator
 - disassembler
 - LEM1802 monitor
 - execution by step
 - GUI uses [anchovy](https://github.com/MrSmith33/anchovy) library

Planned features:
 - debugger
 - assembler (0xSCA compatible)
 - non-standart devices
